
## 练习3-数字和数学

每种编程语言都有好几种方式来进行数学运算.不要怕:每个程序员都会撒谎说他们是数学天才,但实际上他们并不是.如果他们真是数学天才,他们应该去研究数学,而不是靠编写广告算法和网络游戏来赚钱.

这个练习有许多数学符号.我们立刻对它们命名好让你知道它们都叫什么.你可以一边输入一边叫它们的名字.  
下面就是他们的名字:  

' +   plus'   
' -  minus'  
' /  slash'   
' *  asterisk'  
' %  percent'  
' <  less-than'  
' >  greater-than'  
' <= less-than-equal'  
' >= greater-than-equal'
