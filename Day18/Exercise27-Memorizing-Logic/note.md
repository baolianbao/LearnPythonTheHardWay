### 术语
在 Python 中我们会用到下面的术语来定义事物的真` True`或假` False` 计算机的逻辑就是在程序的某个位置检查这些字符或者变量组合在一起的结果是真还是假.  
* and 与
* or 或
* not 非
* != (not equal)不等于
* == (equal)等于
* '>=' (greater-than-equal)大于等于
* <= (less-than-equal)小于等于
* True 真
* False 假
